"Руководство пользователя Редактор JSON"

1) Пользователь может указать ссылку на файл JSON.
2) Приложение загружает файл, отображает его в удобочитаемом формате, позволяет отредактировать.
3) Пользователь может сохранить отредактированный файл в выбранную директорию файловой системы.
4) Обязательна проверка корректности сохраняемого файла.

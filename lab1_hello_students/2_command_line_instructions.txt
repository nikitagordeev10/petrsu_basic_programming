whoami - печатает логин пользователя
pwd - показывает текущий каталог
clean - очистим экран терминала
ls - печатает содержимое текущего каталога
cd - изменяет текущую директорию
mkdir - создает новый каталог
touch - создает новый файл 
cat - показывает содержимое файла
mv - перемещает файл
cp - копирует файл
rm - удаляет файл
rmdir - удаляет каталог
ps - показывает список запущенных процессов
kill - завершает выполнение процесса

Это только небольшая часть команд, доступных в терминале. 
Дополнительную информацию по конкретной команде, можно найти используя команду 'man'
